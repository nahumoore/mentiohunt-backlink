# Mentiohunt Backlink Article

This repository hosts a simple static article promoting [Mentiohunt](https://mentiohunt.com), a tool that helps founders find backlink opportunities and relevant community mentions for their products.

The main purpose of this project is to get a dofollow backlink from GitHub and promote Mentiohunt through a lightweight GitHub Pages article.

Live page: https://nahumoore.github.io/mentiohunt-backlink/
